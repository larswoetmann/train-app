Funktioner
  • Automatisk valg af nærmeste station
  • Opdatering af afgangstider hvert minut
  • Station kan vælges via dropdown eller query parameter stationId
  • Hver station har en standardretning. Uden for storkøbenhavn er det mod byen, indenfor er det væk fra byen
  • Retning kan skiftes via knap eller query parameter switchDirection=true
  • Man kan lave et bogmærke til en specifik station og retning hvis man ikke bare vil se den nærmeste station
  • Nedtælling for valgt afgang med lyd i sidste 10 minutter. Lyd mutes automatisk hvis afstanden til stationen er mindre end 100 meter. (lyd virker ikke altid hvis skærmen er slukket)
  • Virker fuldt for linje C og delvist for linje B, sig til hvis andre linjer skal tilføjes
